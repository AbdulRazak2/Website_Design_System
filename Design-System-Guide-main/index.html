Colors

My favourite colours, that I intend to use on this project. I equal refer to a color wheel to identify complementary colors. Mostly I will incorporate these primary rules and guidelines that surround color and its use in creating aesthetically pleasing visuals, thereby evoking a particular emotion, vibe, or aesthetic.

Creating visuals that make an impact. Using primary colors as my parent colors, anchoring my design in a general color scheme.

Using color wheel gives me the opportunities to create brighter, lighter, softer, and darker colors by mixing white, black, and gray with the original colors.
